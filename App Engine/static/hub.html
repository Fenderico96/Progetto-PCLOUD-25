Fridge Monitoring System

The Fridge Monitoring System is an intelligent project designed for real-time monitoring of a refrigerator's temperature and door status. Utilizing an Arduino device connected to sensors and a cloud server, the system records data, visualizes it through a graphical interface, and activates alarms (LEDs and a buzzer) in case of anomalies, such as excessively high temperatures or the door being left open for too long.

Arduino® UNO R4 WiFi

Microcontroller with built-in Wi-Fi and Bluetooth® for advanced connected projects and prototyping.

Temperature and Humidity Sensor DHT11

Can measure temperature from 0°C to 50°C with a ±2.0°C accuracy, and humidity from 20 to 80% with a 5% accuracy.

LCD1602 Display

Thanks to its real-time visualization capabilities, every temperature fluctuation or door status change is instantly recorded and displayed on the interface.

Magnetic Door Sensor

A magnetic door sensor that detects changes in the door's state.

Buzzer

Main component of the alarm system, it triggers if the door is left open for more than 10s.

Red LED

Visual component of the alarm system.

Graph

Stores all the data sent by the sensors and plot them.

Remote controller

Manually triggers the alarm on the Arduino system.
